Django Heroku Connect
=====================

**Django integration Salesforce using Heroku Connect.**

Install
-------

Simply install the PyPi package…

.. code:: shell

    pip install django-heroku-connect

…and add ``heroku_connect`` to the ``INSTALLED_APP`` settings.
